Coffe Show

Você está desenvolvendo um site que integra uma API externa para o cadastro de variedades de imagens café em um banco de dados.

Cadastro Galeria

Cadastrar Imagens

Na seção de cadastro, as variedades de café serão fornecidas de forma aleatória através de uma API integrada. Os usuários terão a opção de visualizar uma nova variedade aleatória ou salvar a variedade no banco de dados. Ao encontrar uma variedade que desejam salvar, poderão clicar em "Salvar" e adicionar um título para a variedade. Essas variedades serão então armazenadas no banco de dados local.

Acessar o Cadastro

Galeria COFFE

A galeria exibirá as variedades de café que foram salvas pelos usuários. Será organizada em um layout de grade, permitindo aos usuários navegar visualmente por suas coleções.
